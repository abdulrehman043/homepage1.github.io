How to convert your website into an android app ?

Overview:

When you've created your website you also want to publish it as an android app on the playstore or whatever platform you like. Because, most of the world population is using a smartphone, it's the truth that the future of web surfing is totally on the mobile view. So in order to gain more traffic to your website an android app is the best way because to be honest we don't like to go to browser and type your web address(especially on the phone). So, let's get started in building app for your website.

Softwares you need to install:

Android studio is where you build yor app, you can download it from HERE.


So, after downloading --> Installing, open the Android studio adn create a new project


Now after opening a new project select a blank activity


Then fill out the following information i.e your app name,location, etc. And for Minimum SDK select API 17:Android Jelly bean 4.2( it allows your app to run on as many devices) adn click "Finish"


Now, a file opens in front of you, this is your activty_main.xml file. On this you design your app's front end by drag and drop components from the pallete section.



On the adjacent side you'll find your MainActivty.java file here you write your app's backend code


Now, In the left you'll find a pallete section click on th widgets, and select Webview.


Drag the webview widget into your main activity and cover it on your screen by scaling as shown


This is all your front-end work. Now coming to the backend or your MainActivty.Java file copy the code as shown below.

After pasting the code in your MainActivity.Java file save it. And then plug your smartphone on your system via a USB and go to settings and enable USB debugging. After that the Android studio will recognize it as shown below .


On the right you'll see a "run app" button click it and it will install your app on your smart phone. If it succesfully run as you desire then In the Android menu, go to Build --> Build Bundle(s) / APK (s) --> Build APK(s). Android Studio will start building the APK for you....
